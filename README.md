# Urban–rural TB prevalence meta-analysis

Code and outputs for an urban–rural tuberculosis prevalence meta-analysis and associated prediction work.

## Repository structure
- `code/` helper functions
- `data/` input datasets
- `output/` figures/tables generated by the analysis
- `1_urbanMetaAnalysis.Rmd` main analysis
- `2_urban-rural-tb-preds_withEM.qmd` predictions / extensions

## Reproducibility
1. Clone the repository
2. Open the project in R / Positron
3. Install required packages (optionally via renv)
4. Render the analysis documents:
   - `1_urbanMetaAnalysis.Rmd`
   - `2_urban-rural-tb-preds_withEM.qmd`
